# Software / Hardware Bifurcation in AI

Across April 14-26 briefs, the market has consistently rewarded AI hardware (semiconductors, networking, power) over AI software (cloud SaaS, application software). With long-end yields sticky around 4.30-4.34% and credit (LQD) below trend, multiple-expansion mechanics favor capex-backed earnings durability over long-duration application valuations.

## Why It's Happening

1. **Rates regime:** 10Y back at 4.34% (4/23) after compressing in mid-April. Long-duration software multiples discount more aggressively at higher rates.
2. **Earnings impulse:** Hardware (INTC +31% on 4/24, AMD/MU narratives, SMCI/DELL/HPE supply tightness) is delivering pricing power. Software lacks comparable beat-and-raise impulse (NFLX guide-down, IGV multiple compression).
3. **Capex commitment:** AMZN/GOOGL/MSFT mega-deals lock in hardware spend, making semi revenue more predictable. Software faces ad/consumption-cycle uncertainty.
4. **AI monetization split:** Hardware captures clear AI dollars (GPUs sold). Software has to prove AI-native revenue (Copilot attach, AI services), which is murky in current prints.

## Trade Implications

**Long:** SMH, NVDA, AMD, MU, INTC, AVGO, ASML, TSM (high-confidence cash flows)
**Short / underweight:** IGV broadly, generic SaaS, lower-quality software momentum names
**Neutral with caveat:** MSFT, GOOGL — nominally "software" but functionally hyperscaler hardware/capex plays; treat as hardware beneficiaries.

## What Would Reverse This

- 10Y yields rolling to ~3.80% or lower → multiple expansion for software duration
- Software earnings beat-and-raise cycle (ServiceNow, Salesforce, Snowflake) → re-rating
- Hardware capex digestion phase → demand normalization → ASIC/GPU multiple compression
- Anthropic/OpenAI-style monetization milestones → AI software revenue inflection
